# Smart-study-planner
It is a simple program that allocates study time to different courses based on difficulty level and current progress. 


#Features
- It takes number of courses as input
- Collects name of course, difficulty level and progress
- Allocates time based on priority
- Displays a structured study plan
- It also includes input validation and error handling

#How it works
1. User inputs courses and study time
2. Program calculates priority scores
3. Time is distributed based on priority
4. Final study plan os displayed
